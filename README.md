# Student Result Management System

A desktop application developed using Python and SQLite for managing student academic records and examination results.

The system allows administrators to add students, manage courses, enter examination results, and generate reports. Students can view their results through a user-friendly interface.

## Features

- Student Registration
- Course Management
- Result Entry and Management
- Student Dashboard
- Secure Login System
- Result Search and Viewing
- Database Integration using SQLite

## Technologies Used

- Python
- Tkinter
- SQLite3

## Project Structure

- login.py – Login module
- dashboard.py – Admin dashboard
- student.py – Student management
- course.py – Course management
- result.py – Result management
- ResultManagementSystem.db – Database
